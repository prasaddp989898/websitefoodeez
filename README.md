
# Foodeez

Welcome to Foodeez! This is a modern food delivery web app built with React and Vite. Order from top restaurants, track your delivery in real-time, and enjoy a beautiful, fast user experience.

## Features

- Browse restaurants and menus
- Add items to your cart
- Checkout with UPI or Cash on Delivery
- Real-time order tracking
- Responsive design for mobile and desktop

## Getting Started

### Prerequisites

- Node.js (v18 or above recommended)

### Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. (Optional) Create an `.env.local` file for environment variables

### Running Locally

Start the development server:

```bash
npm run dev
```

Open your browser and go to [http://localhost:3002](http://localhost:3002)

### Build for Production

```bash
npm run build
```

## Project Structure

- `src/` — Main source code
- `public/` — Static assets
- `index.html` — Entry point
